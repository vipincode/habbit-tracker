## Dependency

```terminal
npm install express cors dotenv helmet compression morgan
```

## Dev Dependency

```terminal
npm install -D typescript tsx @types/express @types/node @types/cors @types/morgan @types/compression @types/helmet
```

**Step 1.** Install Dependencies custom-env

```terminal
npm install custom-env zod
npm install -D @types/node
```

## Install Mongoose

```terminal
npm install mongoose
npm install -D @types/mongoose
```

## Install RateLiming

```terminal
npm install express-rate-limit
npm install -D @types/express-rate-limit
```
